⚙️ Firebase Setup

Step 1 — Firebase Project Banao

1. console.firebase.google.com kholo
2. "Add project" → naam: court-baghpat
3. Google Analytics: OFF → Create project
4. Left menu: Build → Realtime Database
5. "Create Database" → Asia-Southeast → Test mode
6. Left menu: Project Settings (⚙) → Your apps → Web (🌐)
7. App nickname: court → Register app
8. firebaseConfig copy karo neeche

Step 2 — Config Paste Karo

⚠️ Database Rules: Firebase Console → Realtime DB → Rules tab mein paste karo:
{"rules":{".read":"true",".write":"true"}}
(Baad mein security badha sakate ho)
